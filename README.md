# 🚀 Smart Assignment & Internship Management System (SAIMS)

## 📌 Project Overview

SAIMS is a full-stack web application designed to manage:

* 📚 Student assignments
* 📝 Internship logs
* 👨‍🏫 Teacher evaluations
* 📊 Admin reports

This project is built as a **real-world team collaboration system**, simulating how software companies work.

---

## 👥 User Roles

### 👨‍🎓 Student

* Register & Login
* Submit assignments
* Upload internship logs
* View feedback & marks

### 👨‍🏫 Teacher

* View submissions
* Approve / Reject work
* Give marks & feedback

### 👨‍💼 Admin

* Manage users
* Create assignments
* View reports & analytics

---

## 🔁 Workflow (IMPORTANT)

This project follows a **branch + pull request workflow**.

### ❗ Rules:

* ❌ Do NOT push directly to `main`
* ✅ Always create a new branch
* ✅ Always create a Pull Request (PR)

---

## 🪜 Development Steps

### 1. Clone Repository

```
git clone https://github.com/saims-tech/saims-project.git
cd saims-project
```

---

### 2. Create Branch

```
git checkout -b feature-name
```

Example:

```
git checkout -b frontend-login
```

---

### 3. Add Changes

```
git add .
git commit -m "Your message"
```

---

### 4. Push Code

```
git push origin feature-name
```

---

### 5. Create Pull Request

* Go to GitHub repo
* Click **Compare & Pull Request**
* Submit PR
